# فرص خضراء (Foras Khadra)

أداة تصنيف وتلخيص فرص ومنح باستخدام الذكاء الاصطناعي.

## الفكرة
يقوم المستخدم بلصق نص وصف فرصة أو منحة، وتقوم الأداة تلقائياً بـ:
- تلخيص النص
- تصنيف نوع الفرصة
- استخراج كلمات مفتاحية (tags)

## التقنيات المستخدمة
- Backend: Python + Flask
- AI Model: meta-llama/Llama-3.1-8B-Instruct عبر Hugging Face Inference API
- Frontend: HTML (RTL) بتصميم عربي بألوان أخضر/كريمي/ذهبي

## طريقة التشغيل
1. تثبيت المكتبات: `pip install -r requirements.txt`
2. إضافة مفتاح Hugging Face في ملف `.env` بمتغير `HF_TOKEN`
3. تشغيل السيرفر: `python app.py`

## كيف تعمل الأداة
يستقبل السيرفر نص الفرصة، يرسله إلى نموذج الذكاء الاصطناعي عبر Hugging Face، ويعرض النتيجة (ملخص، تصنيف، tags) للمستخدم بواجهة عربية.
